## 题目

给你一个整数数组 `arr` 。请你将数组中的元素按照其二进制表示中数字 **1** 的数目升序排序。

如果存在多个数字二进制中 **1** 的数目相同，则必须将它们按照数值大小升序排列。

请你返回排序后的数组。

<!--more-->

**示例 1：**

```
输入：arr = [0,1,2,3,4,5,6,7,8]
输出：[0,1,2,4,8,3,5,6,7]
解释：[0] 是唯一一个有 0 个 1 的数。
[1,2,4,8] 都有 1 个 1 。
[3,5,6] 有 2 个 1 。
[7] 有 3 个 1 。
按照 1 的个数排序得到的结果数组为 [0,1,2,4,8,3,5,6,7]

```

**示例 2：**

```
输入：arr = [1024,512,256,128,64,32,16,8,4,2,1]
输出：[1,2,4,8,16,32,64,128,256,512,1024]
解释：数组中所有整数二进制下都只有 1 个 1 ，所以你需要按照数值大小将它们排序。

```

**提示：**

- `1 <= arr.length <= 500`
- `0 <= arr[i] <= 10^4`

## 思路

非常朴素的一种想法是分别算出 arr 中每个数字对应的1的个数，然后用java的函数式接口设定排序规则

对应代码

```java
class Solution {
        public int[] sortByBits(int[] arr) {
            int len = arr.length;
            if (len == 0) {
                return arr;
            }
            List<Item> items = new ArrayList<>();
            for (int i = 0; i < len; i++) {
                Item item = new Item(arr[i], Integer.bitCount(arr[i]));
                items.add(item);
            }
            Collections.sort(items, new Comparator<Item>() {
                @Override
                public int compare(Item o1, Item o2) {
                    return o1.count == o2.count ? (o1.value - o2.value) : (o1.count - o2.count);
                }
            });
            for (int k = 0; k < len; k++) {
                arr[k] = items.get(k).value;
            }
            return arr;
        }

        class Item {
            int value;
            int count;

            public Item(int value, int count) {
                this.value = value;
                this.count = count;
            }
        }
    }
```

翻看评论区之后发现一种很棒的新思路，眼前一亮。

我们对数据进行大小排序的时候，其实是在对**某一个选定的权值**进行排序。

对于本题，占大比重的权值就是**数字对应的二进制数中1的个数**，次要的权值是这个数本身的大小，所以，我们可以通过凸显主要权值的方式进行排序。

对应代码

```java
public int[] sortByBits(int[] arr) {
            int weight = 1000000;
            int len = arr.length;
            if (len == 0) {
                return arr;
            }
            for (int i = 0; i < len; i++) {
                //二进制中1的个数乘了一个足够大的比重 数字本身的大小作为次要权值加上
                arr[i] = weight * Integer.bitCount(arr[i]) + arr[i];
            }
            Arrays.sort(arr);
            //恢复arr本来的数字
            for (int j = 0; j < len; j++) {
                arr[j] %= weight;
            }
            return arr;
        }
```
