## 题目

给定一个包含 0 和 1 的二维网格地图，其中 1 表示陆地 0 表示水域。

网格中的格子水平和垂直方向相连（对角线方向不相连）。整个网格被水完全包围，但其中恰好有一个岛屿（或者说，一个或多个表示陆地的格子相连组成的岛屿）。

<!--more-->

岛屿中没有“湖”（“湖” 指水域在岛屿内部且不和岛屿周围的水相连）。格子是边长为 1 的正方形。网格为长方形，且宽度和高度均不超过 100 。计算这个岛屿的周长。

[![Btdakj.png](https://s1.ax1x.com/2020/10/30/Btdakj.png)](https://imgchr.com/i/Btdakj)

## 思路

由于每块陆地有四条边,所以本题实际是求,每个正方形中有几条边可以纳入最终的周长之中。

据此,可将所有的陆地分为两类

1. 位于矩形边界处,此时其与边界重合的那些边一定可以纳入最终的周长
2. 位于矩形内部,判断其四周哪个方向没有陆地,将这个方向的边纳入周长

要注意的是,对于第1种情况,还需要判断该陆地的内侧是否为水域,如果是,则对应的内侧边也要纳入周长

## 代码

```java
class Solution {
    public int islandPerimeter(int[][] grid) {
        int ans = 0;
        for(int i = 0; i < grid.length; i++){
            for(int j = 0; j < grid[0].length; j++){
								//判断每个正方形的4条边是否能纳入周长
                if(grid[i][j] == 1){
                    if(i == 0 || grid[i-1][j] == 0){
                        ans++;
                    }
                    if(j == 0 || grid[i][j-1] == 0){
                        ans++;
                    }
                    if(i == grid.length-1 || grid[i+1][j] == 0){
                        ans++;
                    }
                    if(j == grid[0].length-1 || grid[i][j+1] == 0){
                        ans++;
                    }
                }
            }
        }
        return ans;
    }
}

```
