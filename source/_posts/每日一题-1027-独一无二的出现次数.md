## 题目
**给你一个整数数组 arr，请你帮忙统计数组中每个数的出现次数。**

**如果每个数的出现次数都是独一无二的，就返回 true；否则返回 false。**

<!--more-->

## 思路
1. 将数组排序
2. 通过比较相邻元素是否相等，统计元素的出现次数
3. 如果当前次数count能加入set，说明是独一无二的，否则说明之前已经有数字出现了count次

### 注意
只有两个元素时需要特殊处理，否则会略过 ```[a,b]``` 即两个不相等元素皆出现一次的情况
# 代码

```java
    class Solution {
        public boolean uniqueOccurrences(int[] arr) {
            int len = arr.length;
            if (len == 2 && arr[0] != arr[1]) {
                return false;
            }
            Arrays.sort(arr);
            int count = 1;
            Set<Integer> set = new HashSet<>();
            for (int i = 1; i < len; i++) {
                if (arr[i - 1] == arr[i]) {
                    count++;
                } else {
                    if (set.add(count)) {
                        count = 1;
                    } else {
                        return false;
                    }
                }
            }
            return true;
        }
    }
```
