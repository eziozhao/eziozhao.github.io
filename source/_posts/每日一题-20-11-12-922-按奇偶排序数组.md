## 题目

给定一个非负整数数组 `A`， A 中一半整数是奇数，一半整数是偶数。

对数组进行排序，以便当 `A[i]` 为奇数时，`i` 也是奇数；当 `A[i]` 为偶数时， `i` 也是偶数。

你可以返回任何满足上述条件的数组作为答案。

<!--more-->

**示例：**

```
输入：[4,2,5,7]
输出：[4,5,2,7]
解释：[4,7,2,5]，[2,5,4,7]，[2,7,4,5] 也会被接受。

```

## 思路

见代码

## 代码

```java
class Solution {
        public int[] sortArrayByParityII(int[] A) {
            int len = A.length;
            int[] B = new int[len];
            int odd = 1, even = 0;
            for (int i = 0; i < len; i++) {
                if (A[i] % 2 == 0) {
                    B[even] = A[i];
                    even += 2;
                } else {
                    B[odd] = A[i];
                    odd += 2;
                }
            }
            return B;
        }
    }
```

### 如果要求空间复杂度为O(1) 可以使用下面的代码

```java
class Solution {
    public int[] sortArrayByParityII(int[] A) {
        int len = A.length;
        int odd = 1, even = 0;
        while(odd < len && even < len){
            if(A[even] % 2 != 0){
                while(A[odd] % 2 == 1){
                    odd += 2;
                }
                int temp = A[odd];
                A[odd] = A[even];
                A[even] = temp;
            }
            even += 2;
        }
        return A;
    }
}
```
