## 题目

我们有一个由平面上的点组成的列表 `points`。需要从中找出 `K` 个距离原点 `(0, 0)` 最近的点。

（这里，平面上两点之间的距离是欧几里德距离。）

你可以按任何顺序返回答案。除了点坐标的顺序之外，答案确保是唯一的。

**示例 1：**

```
输入：points = [[1,3],[-2,2]], K = 1
输出：[[-2,2]]
解释： 
(1, 3) 和原点之间的距离为 sqrt(10)，
(-2, 2) 和原点之间的距离为 sqrt(8)，
由于 sqrt(8) < sqrt(10)，(-2, 2) 离原点更近。
我们只需要距离原点最近的 K = 1 个点，所以答案就是 [[-2,2]]。

```

**示例 2：**

```
输入：points = [[3,3],[5,-1],[-2,4]], K = 2
输出：[[3,3],[-2,4]]
（答案 [[-2,4],[3,3]] 也会被接受。）

```

## 思路

### 基本方法

利用java的函数式接口 自定义排序规则 将原数组排序 然后输出前k个元素

### 优化方法

通过改造后的快速排序进行求解

我们进行快速排序时，每一轮都会根据一个分割点a，将原序列分割成两部分。前半部分的元素都大于a，后半部分都小于a。

因此，可以先对数组进行一次快速排序，然后查看分割点a的索引 i，如果a<k，那么我们只找到了数组的前 i 小的元素，还差(k-i)个。所以应该到(i+1,len)这一部分继续寻找分割点。当a>k时，同理。

当a=k时，分割点前的元素正好有k个，问题解决。

## 代码

### 基本方法

```java
class Solution {
        public int[][] kClosest(int[][] points, int K) {
            int len = points.length;
            if (len == 0) {
                return new int[][]{};
            }
            Arrays.sort(points, new Comparator<int[]>() {
                @Override
                public int compare(int[] o1, int[] o2) {
                    return (o1[0] * o1[0] + o1[1] * o1[1]) - (o2[0] * o2[0] + o2[1] * o2[1]);
                }
            });
            return Arrays.copyOfRange(points, 0, K);
        }
    }
```

### 优化方法

参考自[这里](https://leetcode-cn.com/problems/k-closest-points-to-origin/solution/kuai-lai-miao-dong-topkkuai-pai-bian-xing-da-gen-d/)

```java
class Solution {
        public int[][] kClosest(int[][] points, int K) {
            return quickSelect(points, 0, points.length - 1, K - 1);
        }

        private int[][] quickSelect(int[][] points, int low, int high, int index) {
            if (low > high) {
                return new int[][]{};
            }
            //获取分割点索引
            int split = quickSort(points, low, high);
            if (split == index) {
                return Arrays.copyOfRange(points, 0, index + 1);
            }
            return split < index ? quickSelect(points, split + 1, high, index) : quickSelect(points, low, split - 1, index);
        }

        private int quickSort(int[][] points, int low, int high) {
            int[] p = points[low];
            int square = p[0] * p[0] + p[1] * p[1];
            int i = low, j = high + 1;
            while (true) {
                //(1) 最后i j一定会交错 导致j指向的是数组值较小的那一半的最后一个元素
                while (++i <= high && points[i][0] * points[i][0] + points[i][1] * points[i][1] < square) ;
                while (--j >= low && points[j][0] * points[j][0] + points[j][1] * points[j][1] > square) ;
                if (i >= j) {
                    break;
                }
                int[] temp = points[i];
                points[i] = points[j];
                points[j] = temp;
            }
            //根据(1) 将分割点p与数组值较小的那一半的最后一个元素交换
            points[low] = points[j];
            points[j] = p;
            return j;
        }
    }
```
