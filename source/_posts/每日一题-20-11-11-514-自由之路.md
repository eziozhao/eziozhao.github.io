## 题目

电子游戏“辐射4”中，任务“通向自由”要求玩家到达名为“Freedom Trail Ring”的金属表盘，并使用表盘拼写特定关键词才能开门。

给定一个字符串 **ring**，表示刻在外环上的编码；给定另一个字符串 **key**，表示需要拼写的关键词。您需要算出能够拼写关键词中所有字符的**最少**步数。

<!--more-->

最初，**ring** 的第一个字符与12:00方向对齐。您需要顺时针或逆时针旋转 ring 以使 **key** 的一个字符在 12:00 方向对齐，然后按下中心按钮，以此逐个拼写完 **key** 中的所有字符。

旋转 **ring** 拼出 key 字符 **key[i]** 的阶段中：

1. 您可以将 **ring** 顺时针或逆时针旋转**一个位置**，计为1步。旋转的最终目的是将字符串 **ring** 的一个字符与 12:00 方向对齐，并且这个字符必须等于字符 **key[i] 。**
2. 如果字符 **key[i]** 已经对齐到12:00方向，您需要按下中心按钮进行拼写，这也将算作 **1 步**。按完之后，您可以开始拼写 **key** 的下一个字符（下一阶段）, 直至完成所有拼写。

**示例：**

```
输入: ring = "godding", key = "gd"
输出: 4
解释:
 对于 key 的第一个字符 'g'，已经在正确的位置, 我们只需要1步来拼写这个字符。 
 对于 key 的第二个字符 'd'，我们需要逆时针旋转 ring "godding" 2步使它变成 "ddinggo"。
 当然, 我们还需要1步进行拼写。
 因此最终的输出是 4。

```

**提示：**

1. **ring** 和 **key** 的字符串长度取值范围均为 1 至 100；
2. 两个字符串中都只有小写字符，并且均可能存在重复字符；
3. 字符串 **key** 一定可以由字符串 **ring** 旋转拼出。

## 思路

- 使用dp[i][j]表示当key的第 i 个字符由ring 的第 j 个字符旋转得到时，所需要的最小步数
- 维护一个数组保存ring的每个字母对应的位置集合
- 拼写key的第i个字母时 需要根据第 i-1 个字母时的状态推算

### 状态转移方程

`dp[i][j]=min(dp[i-1][k]+min(abs(j-k),n-abs(j-k))+1)`


其中，

- k代表key[i-1]所表示的字母在ring中的位置集合
- n代表ring的长度
- abs表示绝对值
- 最后的加1代表按下button的那一步

可以这样理解这个方程：

因为key和ring中的字母都有可能重复，所以key的第i-1个字母在ring中可能有多个对应，同时每个对应需要的步数肯定是不一样的。

dp[i-1][k]则表示了这些对应的步数。

而由i-1到i的过程中，又对应了顺时针和逆时针两种选择，即（j-k)和n-(j-k)

所以拼写第i个字母的最少步数就可以根据所有的拼写第i-1个字母的情况中的最少步数确定。

最后的答案即为dp[len_key-1][i]中的最小值。

## 代码

```java
public int findRotateSteps(String ring, String key) {
            int lenRing = ring.length(), lenKey = key.length();
            List<Integer>[] pos = new List[26];
            //因为不知道key有多少个字母 而且key中字母也可能重复 所以选择常量级别的空间比较高效
            for (int i = 0; i < 26; i++) {
                pos[i] = new ArrayList<>();
            }
            //保存ring的每个字母对应的位置集合
            for (int i = 0; i < lenRing; i++) {
                pos[ring.charAt(i) - 'a'].add(i);
            }
            //dp[i][j] 表示从前往后拼写出key的第 i个字符时
            // ring的第j个字符与 12:00方向对齐的最少步数（下标均从0开始）
            //其实对应的是ring的索引值加1 这里的1是最后按的一下按钮
            int[][] dp = new int[lenKey][lenRing];
            //将步数初始值设为无穷大
            for (int i = 0; i < lenKey; i++) {
                Arrays.fill(dp[i], Integer.MAX_VALUE);
            }
            //状态转移方程涉及了(i-1) 所以这里先把i=0初始化
            // pos[key.charAt(0) - 'a']取出了key[0]这个字母在ring中对应的位置集合
            //之后初始化每一个位置需要移动的步数
            for (int i : pos[key.charAt(0) - 'a']) {
                dp[0][i] = Math.min(i, lenRing - i) + 1;
            }
            //按照状态转移方程进行赋值
            for (int i = 1; i < lenKey; i++) {
                for (int j : pos[key.charAt(i) - 'a']) {
                    for (int k : pos[key.charAt(i - 1) - 'a']) {
                        dp[i][j] = Math.min(dp[i][j], dp[i - 1][k] + Math.min(Math.abs(j - k), lenRing - Math.abs(j - k)) + 1);
                    }
                }
            }
            return Arrays.stream(dp[lenKey - 1]).min().getAsInt();

        }
```
