## 题目

实现获取下一个排列的函数，算法需要将给定数字序列重新排列成字典序中下一个更大的排列。

如果不存在下一个更大的排列，则将数字重新排列成最小的排列（即升序排列）。

必须**[原地](https://baike.baidu.com/item/%E5%8E%9F%E5%9C%B0%E7%AE%97%E6%B3%95)**修改，只允许使用额外常数空间。

<!--more-->

以下是一些例子，输入位于左侧列，其相应输出位于右侧列。

`1,2,3` → `1,3,2`

`3,2,1` → `1,2,3`

`1,1,5` → `1,5,1`

## 思路

获取下一个更大的排列显然需要将原数组后面较大的数字交换到前面。同时，要想变大的程度尽可能小，我们很容易注意到这需要使交换的两个数字尽量靠近右侧。

- 从数组尾部开始，寻找第一对升序的数字，a[i]和a[i+1] , 因为假如从i到尾部都是降序，那么交换之后只能使数字变小
- 从 i+1 开始，找到第一个大于a[i]的数字，并将它与a[i]交换
- 由于处于高位的a[i]已经比原来大，所以 i 之后的数字取最小排列即可满足题意

## 代码

```java
class Solution {
        public void nextPermutation(int[] nums) {
            int len = nums.length;
            if (len == 0 || len == 1) {
                return;
            }
            for (int i = len - 2; i >= 0; i--) {
                if (nums[i] < nums[i + 1]) {
                    for (int j = len - 1; j >= i + 1; j--) {
                        if (nums[j] > nums[i]) {
                            swap(nums, i, j);
                            Arrays.sort(nums, i + 1, len);
                            return;
                        }
                    }
                }
            }
            Arrays.sort(nums, 0, len);
        }

        private void swap(int[] nums, int i, int j) {
            int temp = nums[i];
            nums[i] = nums[j];
            nums[j] = temp;
        }
    }
```
