## 题目

给定一个整数数组 `A`，如果它是有效的山脉数组就返回 `true`，否则返回 `false`。

<!--more-->

让我们回顾一下，如果 A 满足下述条件，那么它是一个山脉数组：

- `A.length >= 3`
- 在 `0 < i < A.length - 1` 条件下，存在 `i` 使得：
    - `A[0] < A[1] < ... A[i-1] < A[i]`
    - `A[i] > A[i+1] > ... > A[A.length - 1]`


**示例 1：**

```
输入：[2,1]
输出：false

```

**示例 2：**

```
输入：[3,5,5]
输出：false

```

**示例 3：**

```
输出：true
```

**提示：**

1. `0 <= A.length <= 10000`
2. `0 <= A[i] <= 10000`

## 思路

先判断数组前半段是否满足**严格升序** 定位到分割点时跳出循环 判断数组后半段是否**严格降序**

## 代码

```java
	class Solution {
        public boolean validMountainArray(int[] A) {
            if (A.length < 3) {
                return false;
            }
            int index = 0;
            while (index + 1 < A.length && A[index] < A[index + 1]) {
                index++;
            }
            //完全升序或完全降序
            if (index == 0 || index == A.length - 1) {
                return false;
            }
            while (index + 1 < A.length && A[index] > A[index + 1]) {
                index++;
            }
            return index == A.length - 1;

        }
    }
```
