## 题目

给定一个以字符串表示的非负整数 *num*，移除这个数中的 *k* 位数字，使得剩下的数字最小。

**注意:**

- *num* 的长度小于 10002 且 ≥ *k。*
- *num* 不会包含任何前导零。

<!--more-->

**示例 1 :**

```
输入: num = "1432219", k = 3
输出: "1219"
解释: 移除掉三个数字 4, 3, 和 2 形成一个新的最小的数字 1219。

```

**示例 2 :**

```
输入: num = "10200", k = 1
输出: "200"
解释: 移掉首位的 1 剩下的数字为 200. 注意输出不能有任何前导零。

```

示例 **3 :**

```
输入: num = "10", k = 2
输出: "0"
解释: 从原数字移除所有的数字，剩余为空就是0。

```

## 思路

- 想要得到更小的数字，应该尽可能从高位移去较大的数字
- 从高位到低位，如果数字变大，那么移去高位后，新的数字高位会更大。反之，移去后高位变小，整体变小
- 用一个队列存储数字字符，每当下一位更大时，直接把这一位加入队列。如果下一位更小，则可把队尾的数字去掉，这样整体就会变小
- 如果数字一直是从左到右升序，那么最后从右向左删除对应数量的元素即可

## 代码

```java
class Solution {
        public String removeKdigits(String num, int k) {
            LinkedList<Character> queue = new LinkedList();
            char[] arr = num.toCharArray();
            int len = arr.length;
            for (int i = 0; i < len; i++) {
                while (!queue.isEmpty() && k > 0 && queue.peekLast() > arr[i]) {
                    queue.pollLast();
                    k--;
                }
                queue.add(arr[i]);
            }
            while (k > 0) {
                queue.pollLast();
                k--;
            }
            while (!queue.isEmpty() && queue.peekFirst() == '0') {
                queue.pollFirst();
            }
            StringBuilder res = new StringBuilder();
            for (char ch : queue) {
                res.append(ch);
            }
            return "".equals(res.toString()) ? "0" : res.toString();
        }
    }
```
