## 题目

给定一个数组 nums，编写一个函数将所有 0 移动到数组的末尾，同时保持非零元素的相对顺序。
<!--more-->

```
示例:

输入: [0,1,0,3,12]
输出: [1,3,12,0,0]
```
## 思路

- 遍历数组将非0元素依次赋值
- 根据索引判断剩下多少个0 然后依次赋值

## 代码
```java
    class Solution {
        public void moveZeroes(int[] nums) {
            int len = nums.length;
            int count = 0;
            for (int i = 0; i < len; i++) {
                if (nums[i] != 0) {
                    nums[count++]=nums[i];
                }
            }
            while (count < len) {
                nums[count++] = 0;
            }
        }
    }
```
