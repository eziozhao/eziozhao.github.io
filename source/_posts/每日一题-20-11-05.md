## 题目

给定两个单词（*beginWord* 和 *endWord*）和一个字典，找到从 *beginWord* 到 *endWord* 的最短转换序列的长度。转换需遵循如下规则：

1. 每次转换只能改变一个字母。
2. 转换过程中的中间单词必须是字典中的单词。

<!--more-->

**说明:**

- 如果不存在这样的转换序列，返回 0。
- 所有单词具有相同的长度。
- 所有单词只由小写字母组成。
- 字典中不存在重复的单词。
- 你可以假设 *beginWord* 和 *endWord* 是非空的，且二者不相同。

**示例 1:**

```
输入:
beginWord = "hit",
endWord = "cog",
wordList = ["hot","dot","dog","lot","log","cog"]

输出: 5

解释: 一个最短转换序列是 "hit" -> "hot" -> "dot" -> "dog" -> "cog",
     返回它的长度 5。

```

**示例 2:**

```
输入:
beginWord = "hit"
endWord = "cog"
wordList = ["hot","dot","dog","lot","log"]

输出: 0

解释: endWord "cog" 不在字典中，所以无法进行转换。
```

## 思路

本题本质为求无向图的单元最短路径问题

- 单词A和B如果只相差一个字母，即可以互相转化，则A和B之间就可看作有一条边
- 从beginWord到endWord的转化过程，其实就是在做广度优先遍历获取最短路径的过程
- 由于初始没有提供图结构，所以相当于一边构建图，一边进行搜索

对于单词A的转化，可以每次都用26个字母中的一个替换A的一位，然后判断A‘是否在字典中，如果存在，则A’和A之间就有一条边。通过这个方法把所有能由A转化得到的单词与A相连，构成图

该思路参考自这位[大佬](https://leetcode-cn.com/problems/word-ladder/solution/yan-du-you-xian-bian-li-shuang-xiang-yan-du-you-2/)，感谢前人做出的贡献

```java
class Solution {
        public int ladderLength(String beginWord, String endWord, List<String> wordList) {
            if (wordList.size() == 0) {
                return 0;
            }
            Set<String> wordSet = new HashSet<>(wordList);
            //用于广度优先遍历的队列
            LinkedList<String> queue = new LinkedList<>();
            //存储已经访问过的单词
            Set<String> visited = new HashSet<>();
            int len = beginWord.length();
            queue.add(beginWord);
            visited.add(beginWord);
            int step = 1;
            while (queue.size() > 0) {
                //因为遍历过程中 size会改变 所以这里先存上本轮的size
                int size = queue.size();
                //广度遍历当前层的节点
                for (int curSize = 0; curSize < size; curSize++) {
                    //取出队首单词
                    String cur = queue.pollFirst();
                    char[] wordArr = cur.toCharArray();
                    for (int i = 0; i < len; i++) {
                        //记录该位置的原始字符
                        char raw = wordArr[i];
                        for (char j = 'a'; j <= 'z'; j++) {
                            if (j == raw) {
                                //这个单词就是原本的单词 直接跳过
                                continue;
                            }
                            wordArr[i] = j;
                            String newWord = String.valueOf(wordArr);
                            //转化后的单词位于单词表中才有继续讨论的必要 否则直接进行下一次转化
                            if (wordSet.contains(newWord)) {
                                //转化后已经和endWord相等 转化结束
                                if (newWord.equals(endWord)) {
                                    return step + 1;
                                }
                                //还没得到endWord 将该单词加入队列 并标记为已访问
                                //这些新加入的单词 下一次会作为新的队首被弹出
                                if (!visited.contains(newWord)) {
                                    queue.add(newWord);
                                    visited.add(newWord);
                                }
                            }
                        }
                        //恢复原样
                        wordArr[i] = raw;
                    }
                }
                //这一层没有找到 继续下一轮
                step++;

            }
            return 0;
        }
    }
```
